# Data

This project uses **synthetic data** during the hackathon since live UPI/NPCI
data requires a consent-based data-sharing integration (see the Account
Aggregator / DEPA note in the main README's roadmap section).

- `synthetic/` — generated by `ml/notebooks/generate_synthetic_data.py`.
  Simulates rural transaction seasonality (agricultural cycles, festivals,
  monsoon slack periods) and deliberately includes sparse/irregular history
  for some enterprises, matching the real-world data-completeness problem
  this project is designed around.

Do not commit large generated CSVs — regenerate them locally with the script
instead (see `.gitignore`).
